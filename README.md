# login-and-signup-system
Our secure sign-up module registers new users by enforcing strong password validation and applying cryptographic hashing for data protection. The login system authenticates credentials against the database, issuing time-limited session tokens while concealing specific errors to prevent brute-force attacks.
Project Title
Smart Auth: AI-Powered Login & Sign-Up System
Project Description
Smart Auth is a secure, user-friendly authentication system built and coded within VS Code. It integrates the Google Gemini API to add an intelligent layer to traditional user onboarding. Rather than just processing forms, the system uses AI to provide helpful guidance, interactive feedback, and a smoother, smarter login experience.
Key Features
AI Assistant: Uses Gemini to provide smart feedback on creating secure passwords.
Secure Sign-Up: A robust form that validates user details and safely registers new accounts.
Streamlined Login: A fast, secure gateway that verifies returning users.
Smart Error Handling: Provides clear, helpful guidance when login attempts fail.
Responsive UI: A clean layout built in VS Code that works smoothly on all devices.
Technology Stack
Development Environment: Visual Studio Code (VS Code) for writing and managing the code.
AI Engine: Google Gemini API for driving the smart feedback and interactive elements.
